About

Song Index Application is developed by ETZC Media Team.

It's a simple webpage that provides an interface to access PowerPoint files easily. It's built with a couple of bootstrap code and font awesome icons. The idea is to provide a convenient interface for Media Presenter to quickly access songbook.

This is how it works.

The PowerPoint files were prepared already and saved in a local directory, named as song number respectively. The JSON file contains the song name and number. The webpage gives you a list of song name with their corresponding song number. On clicking a song name, it maps to the corresponding song PowerPoint file with song number and opens it.

It is as simple as that.
